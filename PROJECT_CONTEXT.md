# سياق المشروع

## الهدف
إنشاء منصة Monitoring + Security + Remote Control حقيقية.

## الأهداف التعليمية
- فهم Linux internals
- تعلم Flask Backend
- فهم Networking
- تعلم Cyber Security concepts
- التعامل مع Processes والـ Logs

## التقنيات المستخدمة
- Backend: Python, Flask, Flask-SocketIO
- Frontend: HTML, CSS, JavaScript, Chart.js
- مكتبات: psutil, paramiko, scapy, python-nmap, watchdog

## قواعد التطوير
1. لا نبدأ feature جديدة قبل إنهاء الحالية
2. كل كود جديد في Module منفصل
3. الوظيفة أولاً ثم الشكل
4. عدم نسخ كود بدون فهمه
5. اختبار كل جزء قبل الانتقال للتالي
6. حفظ الأخطاء والحلول في هذا الملف
7. استخدام GitHub لحفظ النسخ

## البيئة
- OS: Kali Linux
- Editor: VS Code
- Python Virtual Environment
